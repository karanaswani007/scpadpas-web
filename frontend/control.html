🤖 ESP32 Integration

Your ESP32 reads the Firestore document devices/device_commands using the Firebase REST API. Include the code below in your ESP32 firmware:

Arduino / ESP32 Pseudocode
// 1. Connect to WiFi
WiFi.begin(SSID, PASSWORD);

// 2. Poll Firestore every 2 seconds
String url = "https://firestore.googleapis.com/v1/"
  "projects/YOUR_PROJECT_ID/databases/(default)/"
  "documents/devices/device_commands"
  "?key=YOUR_WEB_API_KEY";

// 3. Parse JSON response
DynamicJsonDocument doc(1024);
deserializeJson(doc, response);

bool sysEnabled = doc["fields"]
  ["system_enabled"]["booleanValue"];
bool siren      = doc["fields"]
  ["siren"]["booleanValue"];
bool deterrent  = doc["fields"]
  ["deterrent"]["booleanValue"];

// 4. Act on commands
digitalWrite(SIREN_PIN, siren ? HIGH : LOW);
digitalWrite(SPKR_PIN, deterrent ? HIGH : LOW);
📦 Required Libraries:
WiFi.h · HTTPClient.h · ArduinoJson.h
